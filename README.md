## Getting Started

Clone the repository:

```bash
git clone https://github.com/TshegofatsoMkhabela/PLP-portfolio.git
cd PLP-portfolio
```

Install dependencies:

```bash
npm install
```

Run the app:

```bash
npm run dev
```

Open in your browser:

```
http://localhost:5173
```
